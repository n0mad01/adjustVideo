adjustVideo

a video iframe size adjustment jQuery plugin

by Adrian Soluch

Vimeo resize example

$('#section1').adjustVideo(); // simple resizing to the surrounding elements width

Youtube resize example width 450 pixels & centered

$('#section2').adjustVideo({
    'center' : true, // centering the video
    'width' : '450' // resizing video to a specific width ( keeping the aspect ratio )
});

Youtube & Vimeo resize example 400x400

jQuery('#section3').adjustVideo({
    'width' : '400',
    'height' : '400'
});
