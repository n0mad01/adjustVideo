adjustVideo - an video iframe size adjustment jQuery Plugin

by Adrian Soluch

Vimeo video example

                    $('#section1').adjustVideo(); // simple resizing to surrounding elements width

Youtube video example

                    $('#section1').adjustVideo();

Youtube video example

                    $('#section2').adjustVideo({
                        'center' : true, // centering the video
                        'width' : '450' // resizing video to a specific width ( keeping the aspect ratio )
                    });

Youtube video example

                    jQuery('#section3').adjustVideo({
                        'width' : '400', // resizing video to a specific width & height
                        'height' : '400'
                    });
