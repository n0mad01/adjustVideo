adjustVideo

a video iframe size adjustment jQuery plugin

by Adrian Soluch

Simple video resizing plugin. Resizing to surrounding elements width or set width/height properties as you wish, with or without maintaining aspect ratio.

Vimeo & Youtube example

                    
$('#section1').adjustVideo(); // simple resizing to the surrounding elements width

Youtube example width 450 pixels & centered

                    
$('#section2').adjustVideo({
    'center' : true, // centering the video
    'width' : '450' // resizing video to a specific width ( keeping the aspect ratio )
});

Youtube & Vimeo example 400x400

                    
$('#section3').adjustVideo({
    'width' : '400',
    'height' : '400'
});
